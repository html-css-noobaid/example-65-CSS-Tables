CSS Lists, Tables and Forms

CSS and Unordered Lists

  • CSS has several predefined list-style-type property values for unordered lists
  • "disc" is used for bullet points
  • "circle" is used for circles
  • "square" is used for squares
  • "none" is used for removing markers

CSS and Ordered Lists

  1. CSS has several predefined list-style-type property values for ordered lists
  2. "decimal" is for numbers (1, 2, 3, etc.)
  3. "decimal-leading-zero" is for numbers with a leading zero (01, 02, 03, etc.)
  4. "lower-alpha" is for lower case alphabet letters (a, b, c, etc.)
  5. "upper-alpha" is for uppercase alphabet letters (A, B, C, etc.)
  6. "lower-roman" is for lower roman numerals (i., ii., iii., etc.)
  7. "upper-roman" is for upper roman numerals (I, II, III, etc.)
